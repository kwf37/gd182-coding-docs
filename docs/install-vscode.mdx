# Install a Code Editor

If you're tired of typos, or want to try out software engineering tools, follow this tutorial.

We will install VSCode, which is one of the most popular code editors for web development.
It's kind of like the InDesign of code.

## Download VSCode

VSCode is installed like most other desktop programs. Whether you're on Windows or Mac,
they will have instructions on their webpage on how to download and install it.

Visit https://code.visualstudio.com/ and you should see a page similar to the following:

![VSCode Download Homepage](/img/vscode_download_buttons.png)

Look for the download buttons and run the downloaded installer. You should be able to find
Visual Studio code on your computer now!
